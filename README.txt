== Description ==

Connect your WordPress site to the littleBits API and trigger remote actions.

== Installation ==

1. Purchase a CloudBit
2. Login to manage your CloudBit at http://control.littlebitscloud.cc
3. Enter API Key information under Tools > littleBits
